---
title: Installation
description: How to install colophon
---

## From crates.io

```bash
cargo install colophon
```

## From source

```bash
git clone https://github.com/claylo/colophon.git
cd colophon
cargo install --path crates/colophon
```

## Pre-built binaries

Download the latest release for your platform from the
[releases page](https://github.com/claylo/colophon/releases).

## Verify installation

```bash
colophon --version
```
